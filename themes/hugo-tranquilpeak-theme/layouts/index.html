Data Sonification

Three Options for Sonification of Change in Clinical Parameters Prior to Death

This website represents a collection of the sonification of datasets involving various clinical parameters found in dialysis patients in the weeks leading to death. The datasets are divided by the location and sex of the patient. Use the navigation bar to explore the different sonified datasets.

About Data Sonification

Data Sonification is the use of audio to perceptualize data. Whereas data visualization conveys data using visual representations, sonification uses frequency, amplitute, sequences, and time as auditory cues to the listener interpreting the data.

Technologies Used

  • Python
    • Python is a programming language for general-purpose programming.
  • Pandas
    • Pandas is a Python library that allows for powerful data manipulation and analysis, specifically offering useful data structures and operations for analysis. Pandas is particularly effective in dealing with time series data, such as the data presented in the data sets on this site.
  • Chuck
    • ChucK is a strongly-timed programming language for real-time musical composition. It is useful for playing algorithmically-generated musical sequences.
  • Sound Samples
    • All sound samples are courtesy of the Philharmonia Orchestra in London.
